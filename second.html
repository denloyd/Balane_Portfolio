Projects

1. Tile Puzzle Game (C Language)

Tile Puzzle Game

This is a simple Tile Puzzle game made in C language.The goal is to arrange numbered tiles in order from left to right by sliding them into an empty space on the grid. Players move tiles around to get them in the right sequence,from the smallest to largest number.
The game seems easy at first, but it makes you think carefully about every move you make.

2. Point of Sales (C Language)

Point of Sales

This is a simple Point of Sales (POS) system made in C language, designed to help the TUP-M canteen. It can track both inventory and sales, making it easier to manage stock and keep records. The system also has a feature that generates receipts after each transaction.


3. ATM Simulation (C++ Language)

ATM Simulation

This project is an ATM Simulation made in C++Language. The objective of this game is to create a system like an ATM Machine. This system can check balance, withdraw, deposit, etc. Also, this simulation has a feature where you can use your flash drive as
an ATM card to enter the simulation.


4. Simple Database System (Java Language and MySQL)

Simple Database System

This is a simple Database System made using Java and MySQL Workbench. Its purpose is to store user input information in a database. Users can also search, delete, and edit the stored data easily through the system.
